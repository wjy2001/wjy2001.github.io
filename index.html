copy函数

基本认识：

  • 它只能用于切片，不能用于 map 等任何其他类型
  • 它返回结果为一个 int 型值，表示 copy 的长度

坑位一：切片 dst 需要先初始化长度

不是你定义好类型，就能将 src 完全 copy 到 dst 的，你需要初始化长度。

  • 如果 dst 长度小于 src 的长度，则 copy 部分；
  • 如果大于，则全部拷贝过来，只是没占满 dst 的坑位而已；
  • 相等时刚好不多不少 copy 过来。
